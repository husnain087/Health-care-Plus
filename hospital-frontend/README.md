# Hospital Frontend

This is a static frontend structure for a hospital management system. It includes separate sections for patients, doctors, and admin, along with shared assets.

## Structure

```
hospital-frontend/
├── index.html
├── about.html
├── contact.html
├── doctor-search.html
├── patient/
│   ├── login.html
│   ├── dashboard.html
│   ├── profile.html
│   └── appointment.html
├── doctor/
│   ├── login.html
│   ├── dashboard.html
│   ├── schedule.html
│   └── appointments.html
├── admin/
│   ├── login.html
│   ├── dashboard.html
│   ├── approvals.html
│   └── reports.html
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── script.js
│   └── images/
```

Open `index.html` in a browser to get started.
